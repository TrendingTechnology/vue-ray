# vue-ray
## Debug your Vue code with Ray to fix problems faster

This package can be installed in any Vue 2.x or Vue 3.x project to send messages to the [Ray app](https://myray.app).

## Installation

Install with npm:

```bash
npm install vue-ray
```

or yarn:

```bash
yarn add vue-ray
```

### Importing the package

When using in a NodeJS environment (the default), import the package as you would normally:

```js 
// es module import:
import { ray } from 'vue-ray';

// commonjs import:
const ray = require('vue-ray').ray;
```

`npm run test`
